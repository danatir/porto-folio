Do not use the Claude Preview tool to verify or reference UI changes for this project.

**Why:** The preview renders at the wrong screen size, making layout checks unreliable and misleading.

**How to apply:** Make edits and share the localhost URL with the user. Let the user verify visually in their own browser instead.
